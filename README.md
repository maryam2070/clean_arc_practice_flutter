# Posts Application

online/offline application views posts data from API when internet is available, cache it and use cached data when there is no internet
## Features

- Post
    - add post
    - delete post
    - edit Post
    - cache post



## Technologies 
- Dart
- clean archeticture 
- http 
- flutter_bloc
- equatable 
- get_it for dependency injection 
- internet_connection_checker
- shared_preferences
- fluttertoast
